СКАЧАТЬ РАСШИРЕНИЕ ДЛЯ CHROME / YANDEX

При установке расширения пишет ошибку? Вероятно ваш компьютер был заражен. Исправить проблему можно следующим образом:
1. Заходите по адресу c:\Windows\System32\drivers\etc\
2. Открываете файл hosts с помощью блокнота.
3. Ищете строчку 127.0.0.1 clients2.google.com и удаляете ее. Сохраняете изменения в файле.
